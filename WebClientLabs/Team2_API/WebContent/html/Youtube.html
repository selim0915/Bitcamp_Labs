Html의 Youtube API란?

Html에서 Youtube API란, Home Page에서 Youtube의 영상을 제어하기 위한 API이다.
동영상 링크를 가져오기위해선 Youtube의 동영상 소스코드에서 주소를 가져와야 하며, 동영상에는 ID가 있다.


Youtube API의 종류

  • iFrame을 사용한 Youtube 사용
  • iFrame을 사용한 Youtube 자동재생

iFrame을 사용한 Youtebe사용 예제

 본 영상은 HTML의 Youtube API의 기본
 iFrame을 사용해서 가져온 Youtube영상입니다.

 코드 예시 --꺽쇠표시는 생략--

iframe class = "border" style = " padding :10px;" width="420" height="345"
src="https://www.youtube.com/embed/tgbNymZ7vqY"
/iframe

iframe 이란?

  • 하이퍼 텍스트 생성 언어(HTML)
  • 문서에서 글 중의 임의의 위치에 또 다른 HTML 문서를 보여주는 내부 프레임
  • HTML에서 YOUTUBE 영상을 가져오기 위한 요소
  • 개발자가 원하는 크기로 불러들여 보여주는 태그이다.

iframe을 사용한 Youtube 자동재생 예제

 본 영상은 HTML의 Youtube API의 기본
 iFrame autoplay을 사용해서 가져온 Youtube영상입니다.

 코드 예시 --꺽쇠표시는 생략-- ###현재 Youtube측 지원을 하지 않음

iframe class = "border" style = " padding :10px;" width="420" height="345"
src="https://www.youtube.com/embed/tgbNymZ7vqY?autoplay=1"
/iframe

iframe autoplay란?

  • 플레이어가 로드완료 되면 영상을 자동 재생
  • 영상주소에 autoplay = 0 또는 1을 붙인다.
  • 0(기본값) = 플레이어가 로드가 완료돼도 영상을 실행대기한다.
  • 1 = 플레이어가 로드가 완료되면 영상을 자동재생한다.

iframe을 사용한 Youtube Controls 예제

 본 영상은 HTML의 Youtube API의 기본
 iFrame Controls을 사용해서 가져온 Youtube영상입니다.

 코드 예시 --꺽쇠표시는 생략--

iframe class = "border" style = " padding :10px;" width="420" height="345"
src="https://www.youtube.com/embed/tgbNymZ7vqY?controls=0""
/iframe

iframe controls란?

  • 영상주소에 controls=0 또는 1을 붙인다.
  • 0 = 동영상에 재생/정지 버튼이 표시되지 않는다.
  • 1(기본값) = 동영상에 재생/정지 버튼이 표시된다.

iframe을 사용한 Youtube loop 예제

 본 영상은 HTML의 Youtube API의 기본
 iFrame Loop을 사용해서 가져온 Youtube영상입니다.

 코드 예시 --꺽쇠표시는 생략--

iframe class = "border" style = " padding :10px;" width="420" height="345"
src = "https://www.youtube.com/embed/tgbNymZ7vqY
?playlist=tgbNymZ7vqY&loop=1"
/iframe

iframe loop란?

  • 플레이어가 로드완료 되면 영상을 자동 재생
  • 영상주소에 playlist=ID값&loop = 0 또는 1을 붙인다.
  • 0(기본값) = 동영상에 재생이 반복재생하지 않게 한다.
  • 1 = 동영상 재생이 계속 반복되게 한다.
